# Images Folder

## How to Use

### Add Menu Item Images

1. Copy your food images to: `menu/`
2. Name them clearly: `burger.jpg`, `pasta.jpg`, etc.
3. Use in admin dashboard: `/images/menu/burger.jpg`

### Add Category Images

1. Copy category images to: `categories/`
2. Name them: `appetizers.jpg`, `desserts.jpg`, etc.
3. Use in admin dashboard: `/images/categories/appetizers.jpg`

## Image Guidelines

**Recommended:**

- Format: JPG or PNG
- Size: 800x600 pixels or smaller
- File size: Under 500KB
- Names: lowercase with hyphens (e.g., `chicken-burger.jpg`)

**Avoid:**

- Spaces in filenames
- Special characters
- Very large files (>2MB)

## Quick Test

After adding an image, test if it's accessible:

```
http://localhost:5000/images/menu/your-image.jpg
```
